# Group-D-Project-POC

Project refactored into separate frontend and backend folders.

## Structure

```text
Group-D-Project-POC/
  frontend/   React + Vite frontend
  backend/    Placeholder for future backend work
```

## Run the frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

No backend is currently implemented. The backend folder is reserved for future work such as contact form processing, booking integrations, blog/CMS APIs, or admin features.
